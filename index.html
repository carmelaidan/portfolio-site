Automated Growth & Intelligence Engine

Python 3.10+ Streamlit Google Gemini API Serper.dev API FPDF BeautifulSoup & Socket

  • Self-Healing AI Architecture: Engineered a dynamic model selector using Google Gemini that automatically switches between Flash and Pro models to prevent API 404 errors.
  • Deep OSINT & Security Audit: Performs automated SSL verification, high-risk port scanning (FTP, SSH, RDP), and Tech Stack detection (WordPress, React, Shopify).
  • Market Battlefield (Competitor Radar): Integrates the Serper Places API to locate physical business rivals, utilizing aggressive filtering to remove duplicates and broad search fallback for niche categories.
  • Automated PDF Reporting: Generates branded, multi-page audits containing a Digital Health Scorecard, Technical Appendix, and AI-authored executive strategy.
